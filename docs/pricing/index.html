Pricing

RCT - the RV-C Compatibility Tester

You may install RCT on an unlimited number of PCs, but a separate RCT subscription is required for each Omniscope adapter used.

Initial Subscription Annual Renewal
First Adapter Key $1195.00 $395.00
Additional Adapter Keys $395.00 $195.00

You may preview the software for free. The preview allows tests to be run, but no data gets parsed and the tests will not be valid.

Subscriptions include lifetime updates, available via Git or by direct download from https://github.com/TrilliumElectronics/RCT_Public

Omniscope

RCT requires Omniscope. The Omniscope package includes the USB-RVC adapter and cabling required for connecting to your RV-C network.

Omniscope is $495.00, or $375.00 when purchased with a RCT subscription.
